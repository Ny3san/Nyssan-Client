# NYSSAN Client

A premium PvP Client for Minecraft 1.21.11 - Dedicated to Technoblade

## Features
- Crystal PvP, Mace PvP, Anchor PvP modules
- Minimal UI (VAPE-inspired design)
- Macro recording and playback system
- Optimization presets
- Profile system for configurations
- Customizable keybinds

## Setup
1. Run `./gradlew build` to build the mod
2. Find the JAR in `build/libs/`
3. Place it in your `mods` folder with Fabric Loader 0.16.10+

## Keybinds
- **Right Shift** - Open NYSSAN menu
- **[** (Left Bracket) - Open Macro Manager
- All keybinds are customizable in settings

## Modules
### Combat
- Crystal Optimizer - Crystal PvP optimizations
- Mace Enhancement - Mace PvP support
- Anchor Defense - Anchor PvP protection

### Render
- HitBox Display - Show enemy hitboxes
- Armor Durability View - Armor durability percentage
- FPS View - FPS and ping display

### Optimization
- No Camera Shake - Remove camera shake
- No Fog - Remove fog for better visibility
- Entity Culling - Hide invisible entities
